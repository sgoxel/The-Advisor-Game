# The-Advisor-Game
The Advisor Game

Html5 + WebGL Based Game Demo


Game Link: https://sgoxel.github.io/The-Advisor-Game/

Theme: Medieval + Fantasy Roleplaying & Strategy Game with potential LLM AI integration..
You will be peasant at first and will be upgraded to knight lord king emperor.
Since you're only one character in REAL life, In game also you will have no direct control on your sub characters such as armies trade companies.
Instead LLM based AI will decide and move the sub characters. I'm considering to implement free to use online LLM APIs in future.
You will decide policies, international relations, diplomacies, making wars, building cities, castles, making armies like a king!

The game will be light weight and it must work online or offline. Game characters will be 2.5D and AI generated. I will focus on game mechanics more then characters for very long time..
